清除浏览器数据

清除浏览器的本地存储数据，系统将重新进入首次设置模式。这是最简单的重置方法。

立即重置（危险）

立即清除所有数据并重置系统。警告：这将删除所有博客数据和设置！

手动清除数据步骤：

  1. 按 F12 打开浏览器开发者工具
  2. 切换到 Application 或 存储 标签页
  3. 在左侧找到 Local Storage
  4. 选择当前网站的域名
  5. 删除以下键值：
    • blog_auth_settings
    • blog_session
    • blogData（可选，会删除所有博客内容）
  6. 刷新页面，系统将进入首次设置模式
返回登录页面
